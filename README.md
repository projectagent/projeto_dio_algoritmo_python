# 🚀 Resolvendo Códigos em Python com IA

Este repositório foi criado para documentar a resolução de desafios práticos de lógica de programação utilizando **Python**, contando com o suporte de ferramentas de Inteligência Artificial (como GitHub Copilot e modelos locais de LLM) para otimização de código, refatoração e aprendizado contínuo.

O objetivo principal é exercitar a sintaxe da linguagem, estruturar algoritmos eficientes e explorar o papel das IAs como parceiras críticas no processo de desenvolvimento de software.

---

## 🛠️ Tecnologias & Ferramentas
* **Linguagem:** Python 3.x
* **Assistentes de IA:** GitHub Copilot / Gemini 
* **Ambiente de Desenvolvimento:** VS Code / Terminal

## 🧩 Desafios Inclusos
* **Concatenador de Dados:** Manipulação de strings e entradas de usuário.
* **Repetidor de Textos:** Estruturas de repetição e multiplicação de strings.
* **Operações Matemáticas Simples:** Operadores aritméticos fundamentais.
* **Verificador de Números Pares/Ímpares:** Estruturas condicionais.
* **Cálculo de Média:** Manipulação de listas e operadores de divisão.
* **Identificador de Palíndromos:** Fatiamento (slicing) de strings e lógica reversa.
